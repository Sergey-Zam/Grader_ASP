Назначение программы:
Веб-приложение Grader предназначено для сравнения сборок и деталей, созданных в Autodesk Inventor.

Требования к ПО:
• ОС Windows;
• необходимо, что бы на компьютере был установлен Autodesk Inventor (желательно последней версии);
• Visual Studio (желательно последней версии), с пакетом разработки веб-приложений ASP.NET (visual basic);
• Интернет-браузер (желательно Google Chrome).

Запуск программы:
Проект находится на стадии разработки, сейчас для запуска требуется Visual Studio (желательно последней версии).
• Открыть файл Grader_ASP.sln с помощью Visual Studio;
• начать отладку (клавиша F5).
Первый запуск может быть долгим, и занять до 5 минут. Запуск может быть ускорен, если Inventor уже будет запущен. Если Inventor не запущен, программа запустит его сама.

Алгоритм работы:
1. Выбрать один из двух режимов работы: сравнить сборки, или сравнить детали.
2. Выбрать эталонный файл (образец) и проверяемый файл. Для сборок это файл (.iam), для деталей (.ipt).
3. Нажать кнопку "Загрузить файлы на сервер и получить результат".
4. Подождать, первое сравнение может продлиться продолжительное время (для сборок до 5 минут, для деталей до 3 минут).
5. По завершению сравнения, программа выдаст советующее сообщение.
6. Результаты будут представлены в таблице на интерфейсе. Результаты рекомендуется сразу экспортировать в файл MS Excel. Имя экспортированного файла будет начинаться с «Results from grader…» плюс текущие дата и время.

Возможные проблемы:
• При открытии экспортированного файла MS Excel может возникнуть сообщение, что формат файла не советует расширению файла. В диалоговом окне необходимо выбрать "Да" – открыть файл.
• Слишком долгое сравнение документов. Откройте (перейдите к) Autodesk Inventor. Скорее всего, Autodesk Inventor открыл собственное окно, и ожидает ответа пользователя. Из за этого, Grader приостановил работу. Пример ситуации: до начала работы с Grader в Inventor были открыты какие-то документы, и Inventor при попытке Grader их закрыть ожидает ответа, сохранить ли внесенные в них изменения? Как только вы завершите работу с окном, Grader автоматически продолжат работу дальше. Возможны и другие диалоговые окна Inventor, ожидающие ответа пользователя.
• Не сравниваются чертежи, «привязанные» к деталям. Для корректной работы необходимо, чтобы имя чертежа совпадало с именем детали, к которой он относится. Например, деталь имеет имя «crossbar.ipt». Тогда, ее чертеж должен иметь имя «crossbar.idw».
